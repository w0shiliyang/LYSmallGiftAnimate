# LYSmallGiftAnimate
小礼物动画

demo特点：

- [x] 连击规则是根据comboId来区分，一样则表示正在连击。
- [x] 自己送的礼物会优先显示，被挤掉的显示中的礼物在自己送完礼物后优先显示。
- [x] 单线程接收礼物，单线程取，不会有线程问题，项目上可以在子线程接收礼物。
